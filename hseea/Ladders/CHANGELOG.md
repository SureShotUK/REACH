# Changelog - Ladders Project

All notable changes to the Ladders compliance documentation will be documented in this file.

## [Unreleased] - 2025-12-04

### Added
- New subsection "Retrospective Application of Design Standards" in Section 3 of Emergency_Escape_Ladder.md (lines 142-257)
- Comprehensive guidance on when BS 4211:2005+A1:2008 applies to existing vs new installations
- Clear distinction between grandfathered pre-2006 installations and post-2005 requirements
- Example scenarios table showing when current standards apply vs don't apply
- Documentation requirements section for pre-2006 ladders
- Guidance on substantial modifications vs minor maintenance

### Changed
- Updated "Design and Structural Compliance" checklist in Section 11 (lines 796-822)
- Reorganized compliance checklist into three categories:
  - For Ladders Installed Post-2005
  - For Existing Ladders Installed Pre-2006
  - All Ladders (Regardless of Installation Date)

### Documentation
- Clarified that design standards apply from date of installation, not retrospectively
- Documented ongoing safety obligations that apply regardless of installation date
- Added clear triggers for when existing ladders must comply with current standards
- Provided practical guidance for structural engineer discussions during 5-yearly surveys
- Updated ../CLAUDE.md with "Retrospective Application of Standards" principle for broader HSE compliance guidance
