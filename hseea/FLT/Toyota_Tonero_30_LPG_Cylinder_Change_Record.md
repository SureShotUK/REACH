## Procedure Checklist

Tick each step as completed, in order. Do not skip any step.

| Phase | Step | ‚úì |
|---|---|---|
| **Shutdown** | Outflow valve closed (clockwise); engine run dry; fuel switch OFF; ignition OFF; key removed | |
| **Shutdown** | Engine cooled minimum 5 minutes | |
| **Remove** | Pipe valve closed (clockwise) before disconnecting piping | |
| **Remove** | Piping disconnected (counter-clockwise); set pin pulled; cylinder removed | |
| **Remove** | Empty cylinder moved to designated storage area immediately | |
| **Install** | Replacement cylinder inspected (Section above completed) | |
| **Install** | Bands set to correct diameter; set pin locked securely | |
| **Install** | Piping connected firmly to service valve; pipe valve open | |
| **Leak test** | Outflow valve opened (counter-clockwise); ignition ON; fuel switch cycled ON/OFF several times, left OFF | |
| **Leak test** | Soapy water applied to all connections ‚Äî **no bubbles observed** | |
| **Leak test** | Regulator fuel test bar pressed; soapy water applied to regulator connections ‚Äî **no bubbles observed** | |
| **Leak test** | All soapy water wiped off thoroughly | |
| **Return** | No smell of gas before starting engine | |
| **Return** | Engine started normally | |
