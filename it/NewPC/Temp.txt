Windows Claude Code MCP configuration
======================================
Add this to: C:\Users\Steve\.claude\.mcp.json

If that file already exists, merge the "rag" block into the existing "mcpServers" object.
If the file doesn't exist yet, create it with the full content below.

IMPORTANT: Before doing this, set PGPASS as a Windows User environment variable:
  1. Open Start, search "Edit the system environment variables"
  2. Click "Environment Variables..."
  3. Under "User variables for Steve", click "New"
  4. Variable name:  PGPASS
  5. Variable value: [your PostgreSQL password - same value as on Amelai]
  6. Click OK, then OK again
  7. Restart Claude Code after setting it

--- NOTES ---
* PGPASS is read from the Windows User environment variable you set above.
* Knowledge base names will show as UUIDs on Windows (e.g. 1c973c09-...) because
  the docker exec name lookup only runs locally on Amelai. Search results are
  still fully correct - only the display label differs.
* If Claude Code cannot find node_modules (unlikely - all packages are pure JS):
  open a Command Prompt, cd to I:\terminai\rag-mcp and run: npm install
* Tailscale must be connected for the tool to reach Amelai.
  If Tailscale ACL restricts port traffic between devices, you may need to add
  allow rules for ports 5432 (PostgreSQL) and 11434 (Ollama) in the Tailscale
  admin console: https://login.tailscale.com/admin/acls
