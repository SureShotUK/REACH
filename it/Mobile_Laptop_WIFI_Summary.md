# Public WiFi Security - Quick Summary
**Last Updated:** November 2025

## Safest Option: Use Your Phone's Hotspot
**🚨 CRITICAL:** Turn OFF WiFi on your phone when using it as a hotspot (use cellular data only)

**Why:** Cellular hotspot is significantly more secure than public WiFi

## If You Must Use Public WiFi
### Before Connecting
- Ask staff at the location for the official network name to ensure you are connecting to the correct network
- Watch for fake "evil twin" networks with similar names such as free_airport or airport_wifi

### Essential Security
- If you can, use a VPN (gives best protection by encrypting all traffic)
- Verify network name with staff before connecting
- Enable Multi-Factor Authentication (MFA) on all important accounts
- Keep all devices and apps updated
- Only visit HTTPS websites (padlock icon in address bar)

### Device Settings
- **💻 Laptop:** Set network to "Public" profile (select "No" for discoverability) Settings -> Network & Internet -> Properties -> Public network
- **📱 iOS:** Enable "Private Address" for WiFi networks
- **📱 Android:** Enable "Use randomized MAC" for WiFi networks

## What to NEVER Do
- Banking or financial transactions
- Shopping with credit cards
- Leave device unattended
- Accept file transfers from unknown devices
- Connect without verifying network name

## Physical Security
- Position screen away from others
- Keep device physically close to you, if not possible then as a minimum lock screen (WIN + L) 
- Turn off Bluetooth when not in use
- Be aware of surroundings

## When Finished
- Disconnect from WiFi
- "Forget" the network to prevent auto-reconnect

## Remember
**❌ Common Myths:**
- HTTPS alone is NOT enough protection
- Reputable locations do NOT have safe WiFi
- Password-protected public WiFi is NOT secure
- Mobile devices are NOT automatically secure

**✅ Key Takeaway:** Use your phone's cellular hotspot whenever possible. If you must use public WiFi, verify the network name.
