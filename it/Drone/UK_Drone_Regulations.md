# UK Drone Regulation & Certification Guide

*Written By:* Steve Irwin  
*Date:* 2026-05-06   
*Regulatory Context:* UK CAA — post-January 2026 rules (Air Navigation Order 2016 as amended)  

---

## Overview

UK drone regulation uses a three-tier framework managed by the Civil Aviation Authority (CAA). Every operator must understand which tier applies to their operation before flying.

| Category | Risk Level | Typical Use |
|----------|------------|-------------|
| **Open** | Low | Recreational flying, basic commercial work, sub-25kg drones within standard rules |
| **Specific** | Medium | Operations that exceed Open Category limits — urban areas, over crowds, BVLOS |
| **Certified** | High | Highly complex ops — large drones (>3m), cargo transport, high-density urban areas |

The vast majority of recreational and commercial drone pilots operate in the **Open Category**. This guide focuses on Open and Specific; Certified is covered briefly as a signpost.

---

## Registration: Flyer ID and Operator ID

Before flying, most pilots need one or both of the following. These replaced the old drone registration system and are managed through the <a href="https://www.caa.co.uk/drones/getting-started-with-drones-and-model-aircraft/registering-to-fly-drones-and-model-aircraft/" target="_blank">CAA drone registration portal</a>.

| | Flyer ID | Operator ID |
|--|----------|-------------|
| **What it is** | Personal competency ID for the person flying | Registration for the person/organisation responsible for the drone |
| **Who needs it** | Anyone flying a drone weighing **100g or more** | Anyone who owns a drone **≥100g with a camera**, or **≥250g** regardless of camera |
| **Cost** | Free | £12.34/year |
| **How to get it** | Online theory test — 40 multiple-choice questions; pass mark 30/40 | Online registration only — no test required |
| **Validity** | 5 years | Annual renewal |
| **Displayed on drone?** | No | Yes — Operator ID must be marked on the drone |

> **January 2026 change:** The Operator ID weight threshold dropped from 250g to **100g** for camera-equipped drones. If you own a sub-250g drone with a camera (e.g. DJI Mini series), you now need an Operator ID.

## Drone Weight Classes

The CAA uses a class mark system (UK0–UK6) based on drone weight. The class mark determines which Open Category subcategory you can fly in. Legacy drones (bought before class marks existed) have their own rules.

| Class | Max Weight | Open Category | Key Notes |
|-------|------------|---------------|-----------|
| **UK0 / C0** | <250g | A1 | Fewest restrictions; can fly over uninvolved people (not crowds); no A2 CofC needed |
| **UK1 / C1** | <900g | A1 | Can fly near (not over) uninvolved people; no A2 CofC needed |
| **UK2 / C2** | <4kg | A2 | Requires A2 CofC to fly in A2 subcategory; otherwise restricted to A3 |
| **UK3 / C3** | <25kg | A3 | Must stay 150m from residential/commercial/industrial/recreational areas |
| **UK4 / C4** | <25kg | A3 | As UK3; must be non-autonomous and fixed wing |
| **UK5 / C5** | <25kg | Specific only | Requires Specific Category authorisation |
| **UK6 / C6** | <25kg | Specific only | As UK5; designed for operations over crowds |
| **Legacy (no class mark)** | <25kg | A1/A3 depending on weight | Pre-class-mark drones: <250g = A1 rules; 250g–25kg = A3 rules only |

More detail on class marks: <a href="https://www.caa.co.uk/drones/getting-started-with-drones-and-model-aircraft/class-marks/" target="_blank">CAA Class Marks</a>

## Open Category

The Open Category covers the majority of drone operations. It is divided into three subcategories based on where and how close to people you fly.

### Subcategory Comparison

| | **A1** | **A2** | **A3** |
|--|--------|--------|--------|
| **Drone classes** | UK0/C0, UK1/C1, legacy <250g | UK2/C2 (with A2 CofC) | UK2–UK4/C2–C4, legacy 250g–25kg |
| **Qualification needed** | Flyer ID | Flyer ID + **A2 CofC** | Flyer ID |
| **Over uninvolved people** | UK0/C0: yes (not crowds); UK1/C1: avoid | No | No |
| **Min. distance from uninvolved people** | UK1/C1: do not fly directly over | 30m; 5m in low-speed/cine mode | Must not intentionally fly over |
| **Distance from populated areas** | No specific requirement | No specific requirement | **150m** from residential, commercial, industrial, recreational areas |
| **Max altitude** | 120m AGL | 120m AGL | 120m AGL |
| **Visual line of sight** | Required | Required | Required |

> **Note:** "Uninvolved people" means anyone not part of your operation. "Populated area" in A3 means you must stay well clear of towns, villages, beaches, busy parks, etc.

Where you can fly reference: <a href="https://www.caa.co.uk/drones/getting-started-with-drones-and-model-aircraft/where-you-can-fly/" target="_blank">CAA: Where You Can Fly</a>

---

### A2 Certificate of Competency (A2 CofC)

The A2 CofC unlocks the A2 subcategory, allowing you to fly UK2/C2 drones closer to people than A3 permits.

| | Detail |
|--|--------|
| **Who needs it** | Pilots flying UK2/C2 drones who want to operate in A2 (within 30m of people) |
| **What it unlocks** | Flying UK2/C2 within 30m of uninvolved people; 5m in certified low-speed/cine mode |
| **Prerequisite** | Valid Flyer ID |
| **How to get it** | Online self-study + theory exam at a CAA-approved Recognised Assessment Entity (RAE) |
| **Typical cost** | £150–£300 depending on RAE provider |
| **Validity** | 5 years |
| **Renewal** | Retest required on expiry; no mid-cycle refresher needed |

Full details: <a href="https://www.caa.co.uk/drones/remote-pilot-qualifications/a2-certificate-of-competence-a2-cofc/" target="_blank">CAA: A2 Certificate of Competency</a>

## Specific Category

When an operation exceeds Open Category limits — flying over crowds, operating in congested urban areas, or BVLOS — it falls into the **Specific Category**. This requires formal CAA authorisation before flying.

Full overview: <a href="https://www.caa.co.uk/drones/specific-category/specific-category-overview/" target="_blank">CAA: Specific Category Overview</a>

---

### PDRA01 — The Simplest Specific Route

**Pre-Defined Risk Assessment 01 (PDRA01)** is a standardised authorisation covering the most common Specific Category operations. It avoids the need for a full bespoke risk assessment.

| | PDRA01 Requirements |
|--|---------------------|
| **Drone weight** | 200g–25kg |
| **Operation type** | VLOS only |
| **Location** | Any UK location (subject to airspace restrictions) |
| **Qualifications** | Remote Pilot must hold a **GVC** (General Visual Line of Sight Certificate) or equivalent |
| **Insurance** | Third-party liability required |
| **CAA authorisation** | Required — submit declaration to CAA |

---

### UK SORA — For Complex Operations

For operations not covered by PDRA01 (e.g. flying over crowds, complex urban environments, or non-standard risk profiles), operators must submit a **UK Specific Operations Risk Assessment (UK SORA)** to the CAA.

- UK SORA replaced the old OSC (Operational Safety Case) method in **April 2025**
- Requires a detailed risk assessment covering ground risk, air risk, and mitigations
- CAA reviews and issues an Operational Authorisation if satisfied
- Typically used by professional operators for high-value commercial work

---

### GVC — General Visual Line of Sight Certificate

The GVC is the standard qualification for Specific Category VLOS operations such as PDRA01.

| | Detail |
|--|--------|
| **What it is** | Theory and practical qualification for Specific Category VLOS ops |
| **Who needs it** | Pilots operating under PDRA01 or other Specific Category VLOS authorisations |
| **How to get it** | Training + theory exam + practical assessment at a CAA-approved RAE |
| **Status** | **Being phased out — GVC will no longer be issued after 31 December 2027** |
| **Replacement** | RPC-L1 (Remote Pilot Certificate Level 1) — see BVLOS section below |

Full details: <a href="https://www.caa.co.uk/drones/remote-pilot-qualifications/level-1-2-and-3-remote-pilot-certificates-and-gvc/" target="_blank">CAA: GVC and Remote Pilot Certificates</a>

## Certified Category

The Certified Category applies to the highest-risk operations and is out of scope for most operators. It is included here for awareness.

**When Certified Category applies:**
- Drones with a characteristic dimension greater than **3 metres**
- Operations transporting people or dangerous cargo
- Operations over assemblies of people where no alternative risk mitigation is possible

**What is required:**
- CAA-certified aircraft (airworthiness approval)
- CAA-licensed remote pilot
- Operator must hold a CAA UAS Operator Certificate (UOC)

This is broadly equivalent to manned aviation regulation and involves significant cost and administrative burden.

## BVLOS — Beyond Visual Line of Sight

All Open Category operations and most Specific Category operations require the pilot to maintain **Visual Line of Sight (VLOS)** — i.e. you must be able to see the drone unaided at all times.

**BVLOS** (Beyond Visual Line of Sight) covers operations where the drone flies beyond the pilot's direct visual range — for example, long-range corridor inspections, pipeline surveys, or autonomous delivery.

### BVLOS Requirements

| | Detail |
|--|--------|
| **Category** | Specific or Certified — BVLOS is **not permitted in Open Category** |
| **Authorisation** | Full UK SORA submission and CAA Operational Authorisation required |
| **Pilot qualification** | RPC-L2 or RPC-L3 (see below) |
| **Complexity** | Significant — requires detailed risk assessment, airspace integration plan, contingency procedures |

### Remote Pilot Certificate Levels for BVLOS

| Certificate | Scope |
|-------------|-------|
| **RPC-L1** | Specific Category VLOS (replaces GVC from 2028) |
| **RPC-L2** | BVLOS in segregated/empty airspace (e.g. over open countryside with airspace reserved) |
| **RPC-L3** | BVLOS in complex/unsegregated airspace (e.g. near aerodromes, urban areas) |

RPC-L2 and L3 are advanced qualifications requiring significant training investment. BVLOS is not a routine add-on — it requires dedicated airspace planning and CAA engagement well in advance of operations.

## Key Flying Restrictions

These apply across all categories unless specific authorisation has been granted.

| Rule | Detail |
|------|--------|
| **Maximum altitude** | 120m (400ft) above ground level |
| **Visual line of sight** | Must keep drone in unaided visual range at all times (except BVLOS authorisation) |
| **Flight Restriction Zones (FRZs)** | No flying within FRZs around licensed aerodromes without ATM permission. Radius typically 5km |
| **Prohibited areas** | Prisons, military sites, nuclear facilities, royal palaces, government buildings |
| **Congested areas** | Open Category A3 must stay 150m from residential/commercial/industrial areas |
| **Emergency services** | Never fly near emergency service operations (police, fire, ambulance, search and rescue) |
| **Temporary restrictions** | Check NOTAMs before every flight — events, military exercises, and incidents trigger TRAs |
| **Night flying** | Permitted in Open Category if drone has adequate lighting; check specific restrictions |

**Tools for checking airspace before flight:**
- <a href="https://www.nats.aero/flight-planning/drone-notam-tool/" target="_blank">NATS Drone NOTAM Tool</a>
- NATS Drone Assist app (iOS/Android)
- <a href="https://www.caa.co.uk/drones/getting-started-with-drones-and-model-aircraft/where-you-can-fly/" target="_blank">CAA AirSpace Explorer</a>

## Recreational vs Commercial Flying

The distinction between recreational and commercial matters primarily for **insurance** requirements. Both can operate in Open Category, but commercial operations often need higher-risk authorisations.

| | Recreational | Commercial |
|--|-------------|------------|
| **Definition** | Flying for personal enjoyment, hobby, sport | Any operation where payment or business benefit is involved — including your own company's inspections |
| **Flyer ID** | Required (drones ≥100g) | Required |
| **Operator ID** | Required (camera drones ≥100g, or ≥250g) | Required |
| **Third-party liability insurance** | Optional (strongly recommended) | **Mandatory** — minimum €750,000 SDR |
| **Typical category** | Open (A1/A3) | Open or Specific depending on operation |
| **PDRA01 / SORA** | Not usually needed | May be required for complex operations |

Insurance requirements: <a href="https://www.caa.co.uk/drones/moving-on-to-more-advanced-flying/insurance-requirements/" target="_blank">CAA: Insurance Requirements</a>

## Quick Decision Guide

Use this table to identify the minimum requirements for your intended operation.

| Drone weight | Camera? | Intended use | Where | Minimum requirements |
|-------------|---------|-------------|-------|----------------------|
| <100g | Any | Any | Any | Nothing required (good practice to follow the Drone Code) |
| 100g–249g | No | Recreational/commercial | Open areas, A3 rules | Flyer ID |
| 100g–249g | **Yes** | Recreational/commercial | Open areas | Flyer ID + **Operator ID** |
| <250g (UK0/C0) | Any | Recreational/commercial | Near people (not crowds) | Flyer ID + Operator ID |
| 250g–900g (UK1/C1) | Any | Recreational/commercial | Near (not over) people | Flyer ID + Operator ID |
| <4kg (UK2/C2) | Any | Recreational/commercial | **Within 30m of people** | Flyer ID + Operator ID + **A2 CofC** |
| <4kg (UK2/C2) | Any | Recreational/commercial | A3 rules (150m from populated areas) | Flyer ID + Operator ID |
| Any ≥200g | Any | **Commercial — over residential/congested** | Urban / over people | Flyer ID + Operator ID + **GVC** + **PDRA01 authorisation** |
| Any | Any | **BVLOS** | Any | Flyer ID + Operator ID + **RPC-L2/L3** + **UK SORA** + CAA Operational Authorisation |
| Any | Any | **Commercial** | Any | As above + **third-party liability insurance** |
